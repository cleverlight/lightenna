Free!

The Lightenna Flash XML slideshow component is currently available for free, in return for a link from your site.

We're trying to build the profile of Lightenna and links back to our site improve our page rank.

Link

The link should read:

Flash XML slideshow by Lightenna

Link hiding: method 1

You can hide the link so long as its visible to the search engines. You can use div.lightenna_link_hook in your CSS to make the link the same colour as the page background.

Link hiding: method 2

The component uses Javascript to overwrite a <div> with code to embed and initialise the slideshow. This div is visible to search engines, but not to your users. By placing the link described above in this div you can help us with our page rank, yet in no way reduce the impact of the slideshow.

L'esprit de corps

This code is offered in a spirit of sharing. Time and effort went into designed and building this component. Please show your appreciation of that by giving us a link.
