3.6.12. Optimise (Limit)

  • Limit
    • Valueable on sorts (0,10 faster than 0,1000!)
    • Beware limit referencing
      • Limit 1000,10 will fetch the first 1000, then give you the 10 you want
        • Search engines
      • Better to use WHERE restriction if possible
