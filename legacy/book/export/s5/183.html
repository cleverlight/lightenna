3.2.04. Normal forms (1NF)

  • First Normal form (1NF)
    • Disallows multivalued attributes
    • Part of the basic relational model
  • Domain must include only atomic values
    • simple, indivisible
  • Value of attribute-tuple in extension of schema
  • t[Ai] ∈ (Ai)
